# 🛡️ TryHackMe Write-Ups

Welcome to my cybersecurity portfolio!

This repository contains detailed write-ups of rooms I have completed on the TryHackMe platform. The goal of this repository is to document my hands-on experience, track my progress, strengthen my reporting skills, and build a professional portfolio as I continue my journey in cybersecurity.

---

## 📚 What's Inside?

Each room has its own dedicated directory containing:

📄 **README.md**

* A complete walkthrough of the room
* Enumeration methodology
* Exploitation process
* Privilege escalation steps
* Lessons learned and key takeaways

📸 **screenshots/**

* Screenshots of important findings
* Evidence of successful exploitation
* Privilege escalation proof
* Interesting discoveries during the engagement

📝 **outputs/**

* Nmap scan results
* Enumeration outputs
* Tool results (Hydra, Gobuster, FFUF, Enum4Linux, etc.)
* Relevant command outputs used throughout the challenge

---

## 🔍 Methodology

While every room is different, most write-ups follow a structured approach:

1. 🔎 Reconnaissance & Enumeration
2. ⚔️ Initial Access
3. 🚀 Privilege Escalation
4. 🏁 Capture Flags / Objectives
5. 📖 Summary & Lessons Learned

---

## 🛠️ Tools & Resources

For convenience, this repository also includes a custom **tools.py** script that automates the installation of the tools I regularly use in my Kali Linux lab environment.

This helps me quickly provision new virtual machines and maintain a consistent toolkit across my learning environments and may assist you.

## ⚠️ Disclaimer

All activities documented in this repository were performed in authorized lab environments provided by TryHackMe for educational purposes only.

No systems outside of these environments were targeted or accessed.
